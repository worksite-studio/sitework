# Reference

External domain material used as input to SITEWORK. Not authored by us; not part of the product spec; cited or quoted only.

## What goes here

- Rawlinson rate sample pages, methodology notes.
- Xero API / connector docs relevant to integration design.
- Australian construction finance conventions (BOQ standards, retention rules by jurisdiction, GST treatment of progress claims, etc.) when they need to live somewhere.
- Anything from external suppliers or industry bodies that informs a design decision.

## What does *not* go here

- Internal spec material → use `docs/finance-model/` or `docs/design/`.
- Code → root or `index.html`.
- Living roadmap / decisions → `ROADMAP.md`.

## Status

Empty as of folder creation. Populate as reference material is gathered.
